# Protótipo - Citinova (Prefeitura de Fortaleza)

## Como rodar
Abra `index.html` no navegador ou use um servidor local.

Exemplo (Python):
```bash
python -m http.server 8000
```
Acesse: http://localhost:8000

## Observações
- Bootstrap via CDN (sem necessidade de baixar arquivos).
- Imagens de Programas e Projetos e links apontam para o site oficial.
- Conteúdo de "Últimas notícias" é placeholder para preenchimento interno.
- Itens Inovafor e Labifor foram removidos do protótipo.
